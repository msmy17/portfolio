Hybrid Letters



This project was to take two different types and create a new typeface out of the combination created from the 2.







The typefaces used to create this hybrid typeface were Dead History and Boucherie Flared. I initially chose Dead History because I saw a good base letterform to start off of to experiment. The stroke of the leterform with the rounded serifs was complimented well by the curves the the Boucherie Flared letters. Disecting the Boucherie Flared forms to create lettern contrast similar to a modern type was a good way to emphasize the strong rounded flares at the end of the letters.
