Video Game Fight Academy



This project was designed to create a new way to treat instructions for a given task using a visual learners approach with mostly motion graphics. The template used was a fighting genre video game





The site was programmed using HTML, CSS, and Bootstrap. The Bootstrap was used in order to create easy to follow guides and columns.






The design was focused on creating an aesthetic similar to a video game guide book, but using full motion graphics, something a paper book could not accomplish. I used guidebooks as an insperation on how to deal with tips that would appear in each section, without feeling intrusive or out of place.
